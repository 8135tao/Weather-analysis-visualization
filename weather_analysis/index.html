Summary

This project utilizes Python (citipy), a simple API (OpenWeatherMap) to study the relationship beween weather and latitude.

By reading a csv file, 1,500 random latitude and longitude across the globe are selected. The Python CitiPy library is then used to find the city closest to that each Lat/Lon (dropping all duplicates) to get a listing of over 500 cities. Finally, the infomation about the weather of each city is gathered using the OpenWeatherMapAPI (dropping cities with no data), and the data are recorded in a dataframe using Pandas, and saved as a csv file.

By using matplolib, the following four relationships are studied, and the visualizations are included in separate pages.

  • Temperature (F) vs. Latitude
  • Humidity (%) vs. Latitude
  • Cloudiness (%) vs. Latitude
  • Wind Speed (mph) vs. Latitude
